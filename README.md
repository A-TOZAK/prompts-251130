# 先生のためのプロンプト集 (Teacher's Prompts)

学校の先生方が授業や校務で使えるAIプロンプトを共有するためのWebサイトです。
モダンで美しいデザインと、シンプルな投稿機能を備えています。

## 機能

- **プロンプト一覧表示**: 投稿されたプロンプトを新しい順に表示します。
- **プロンプト投稿**: タイトル、校種、教科、内容を入力して投稿できます。
- **コピー機能**: ワンクリックでプロンプトをクリップボードにコピーできます。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンで見やすく表示されます。

## 動作環境

- **ブラウザ**: 最新のChrome, Safari, Edge, Firefoxなど
- **ホスティング**: GitHub Pages, Netlify, Vercelなどの静的サイトホスティングサービス

> [!NOTE]
> このサイトは静的HTML/JSで動作します。PHPなどのバックエンドは不要です。

## 投稿機能について

GitHub Pagesなどの静的ホスティングではサーバーサイドのデータベースが利用できないため、**投稿機能は簡易的なもの（LocalStorage保存）**になっています。

- 投稿したプロンプトは、**あなたのブラウザにのみ**保存されます。
- ページをリロードしても消えませんが、キャッシュをクリアすると消えます。
- **他のユーザーにはあなたの投稿は見えません。**

### 全員に公開するプロンプトを追加する方法（管理者向け）

全員に公開したいプロンプトがある場合は、このリポジトリの `data/prompts.json` ファイルを直接編集して、データを追加してください。

## インストール方法

1. このリポジトリをGitHubにアップロードします。
2. GitHubのリポジトリ設定から「Pages」を開き、デプロイ元のブランチ（mainなど）を選択してSaveします。
3. 数分後、発行されたURLにアクセスすればサイトが表示されます。

## ファイル構成

- `index.html`: メインページ
- `style.css`: スタイルシート
- `script.js`: プロンプト読み込み・表示・ローカル保存ロジック
- `data/prompts.json`: 共通プロンプトデータ（JSON）
